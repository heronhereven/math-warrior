# 给小和看的说明

你只需要记住这几步：

1. 双击 `MathQuestXiaohe.exe`
2. 稍等一下，浏览器会自动打开一个本地控制台页面
3. 第一次使用时，在页面里填好：
   - 仓库拥有者
   - 仓库名称
   - Token
   - 勾选“启用 GitHub 同步”
   - 然后点“保存设置”
4. 点“打开小和工作台”
5. 保持这个客户端开着，就可以正常查收泡面侠的提交、审核、回写结果

平时你主要会用到这几个按钮：

- `打开小和工作台`
  打开真正的后台页面
- `立即同步`
  想立刻拉一次新提交、或者立刻把审核结果发回去时点它
- `退出客户端`
  用完再关

如果你打开后发现还没有新内容：

- 先点一次 `立即同步`
- 再回到工作台刷新看看

这套程序的数据会保存在你电脑本地，不会因为关掉浏览器就消失。  
但如果要继续收发新的提交和审核结果，客户端需要保持开着。
